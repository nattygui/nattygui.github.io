Beego安装

// 先安装bee 脚手架
go get github.com/beego/bee

在 go mod 模式下会安装失败，此时需要先关闭该模式再安装bee。

go env -w GO111MODULE=off

安装完成bee脚手架之后，会在gopath目录的bin文件下生成bee可执行文件，将 $GOPATH/bin 路径添加环境变量下。在命令行执行bee，出现以下表明安装bee脚手架成功了。
