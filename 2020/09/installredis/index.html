Centos7 安装redis

1、 检查是否有redis yum 源

若没有执行下一步，若有执行第三步

yum search redis

yum info redis

2、安装 epel 仓库

yum install -y epel-release

3、 安装redis 数据库

yum info redis

yum install -y redis

4、 启动redis服务

systemctl start redis

systemctl enable redis # 设置开机启动

5、进入 redis 客户端

[admin@localhost ~]$ redis-cli
127.0.0.1:6379>
